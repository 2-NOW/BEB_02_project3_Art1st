## Getting Started

dependencies를 설치합니다.

```bash
npm i
```

prisma를 migrate 합니다.

```bash
npx prisma migrate dev
```

```bash
npm run dev
# or
yarn dev
```

sqlite를 사용한 현재 사용중인 local db를 http://localhost:5555 에서 확인할 수 있습니다.
